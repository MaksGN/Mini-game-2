# Міні-гра "Збирач монет" (Лекція 3)

### Що реалізовано:
* Керування гравцем (клавіатура + дотик).
* Збір монет та підрахунок очок.
* Таймер зворотного відліку на 30 секунд.
* Збереження рекорду в `localStorage`.
* Чіткі стани гри: Start, Play, Game Over.

### Модифікації (Домашнє завдання):
* Додано ворога (червоне коло), який переслідує гравця.
* При зіткненні з ворогом — гра закінчується.
* Кожні 5 зібраних монет ворог прискорюється.
* Додано звуковий ефект при зборі монети.

### Що можна додати далі:
* Більше рівнів або перешкод (стіни).
* Різні типи бонусів (наприклад, +5 секунд до таймера).
